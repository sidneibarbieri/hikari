Atalhos de caça

Consultas KQL prontas. Clique para abrir o Discover já filtrado.

  • Últimos 15 minutos Janela curta para reagir a alertas recentes.
  • Respostas HTTP ≥ 400 Erros e falhas de autenticação. Útil para localizar tentativas.
  • Padrão flag no payload Procura a substring no request e na response. Aponta candidatos a indicador.
  • User-Agent automatizado Curl, Python, wget — indicador comum de varredura ou exfiltração.
  • Portas de interesse SSH, RDP, payloads, web alternativa. Útil para movimento lateral.
  • Severidade alta ou crítica Foca nas linhas que o pipeline marcou como prioritárias.
